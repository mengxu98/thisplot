# Convert a plot object to a gtable

Convert various plot objects (gtable, grob, patchwork, ggplot) to a
gtable object.

## Usage

``` r
as_gtable(plot, ...)
```

## Arguments

- plot:

  A plot object (gtable, grob, patchwork, or ggplot).

- ...:

  Additional arguments passed to other functions.

## Value

A gtable object.
