# Convert a plot object to a grob

Convert various plot objects (gList, patchwork, ggplot) to a grob
object.

## Usage

``` r
as_grob(plot, ...)
```

## Arguments

- plot:

  A plot object (gList, patchwork, or ggplot).

- ...:

  Additional arguments passed to other functions.

## Value

A grob object.
