# Build graphics callback list for anno_customize

Build graphics callback list for anno_customize

## Usage

``` r
annotation_graphics(subplots, prefix)
```

## Arguments

- subplots:

  A named list of subplot objects.

- prefix:

  Prefix used to generate stable grob names.

## Value

A named list of callback functions accepted by
[ComplexHeatmap::anno_customize](https://rdrr.io/pkg/ComplexHeatmap/man/anno_customize.html).
